# SandAnalyze - 沙粒形态分析系统设计文档

## 概述

基于 OpenCV 传统图像处理 + YOLOv8-seg 混合方案，对光学显微镜拍摄的沙粒图像进行颗粒识别、形状分析和数量统计，以桌面 GUI 应用形式呈现。

## 背景

- **场景**：地质/沉积学研究
- **图像来源**：光学显微镜拍摄的沙粒图像
- **数据现状**：目前只有一张照片，无标注数据
- **输出形式**：桌面 GUI 应用（PyQt6）

## 架构

## 核心流程

1. 加载图像 → 预处理
2. 双通道检测：传统 OpenCV 轮廓检测 + YOLOv8-seg 推理（可选）
3. 对检测结果计算多维度形态参数
4. GUI 中展示标注图 + 统计图表 + 参数表格
5. 支持导出统计报告

## 图像预处理（core/preprocessor.py）

针对光学显微镜图像的典型处理流程：

1. **灰度化** → 自适应阈值二值化（应对光照不均）
2. **形态学操作** → 开运算去噪 + 闭运算填充小孔
3. **距离变换 + 分水岭** → 分离粘连颗粒
4. **可选**：CLAHE 对比度增强、背景减除

预处理参数可通过 GUI 调节：二值化阈值、最小面积、形态学核大小等。

## 检测模块

### 传统检测（core/traditional.py）

- `cv2.findContours` 提取轮廓
- 过滤面积过小/过大的噪声轮廓
- 返回每个颗粒的轮廓点集和掩码
- **默认检测方法**

### YOLO 检测（core/yolo_detector.py）

- 加载预训练 YOLOv8-seg 模型（yolov8n-seg.pt）
- 推理得到实例分割掩码
- 预训练模型不认识"沙粒"类别，可用 SAM2 自动分割生成伪标注 → 微调
- 作为可选对比视图，不作为默认方法

### 检测结果融合

- 默认使用传统方法结果
- YOLO 结果作为可选对比视图
- 用户可在 GUI 中切换查看

## 形态参数计算（core/morphology.py）

对每个检测到的沙粒轮廓/掩码，计算以下参数：

| 参数 | 计算方法 | 地质意义 |
|------|----------|----------|
| 面积 (A) | 掩码像素数 × 像素面积 | 粒径基础 |
| 周长 (P) | 轮廓像素长度 | 磨蚀程度 |
| 圆度 (Circularity) | 4πA/P² | 越接近1越圆 |
| 等效粒径 (d_eq) | √(4A/π) | 等效圆直径 |
| 长轴 (a) / 短轴 (b) | 最小外接椭圆 | 颗粒延伸方向 |
| 长短轴比 (AR) | a/b | 颗粒扁平程度 |
| 球度 (Sphericity) | b/a（Sneed & Folk） | 三维形状推断 |
| 凸度 (Convexity) | A / A_convex | 表面凹凸程度 |
| Feret 直径 | 最大/最小 Feret | 方向性尺寸 |

### 统计汇总

- 各参数的：均值、标准差、中位数
- 粒径分布直方图（对数正态拟合）
- 圆度-球度散点图
- Zingg 分类饼图（圆盘/棒状/球状/片状）

## GUI 界面（gui/）

### 技术选型：PyQt6

选择 PyQt6 的原因：
- 绘图性能好（大量轮廓叠加显示）
- matplotlib 嵌入方便
- 生态成熟，文档丰富

### 交互功能

- **加载图像**：拖拽或菜单打开
- **检测方法切换**：传统 / YOLO / 对比视图
- **参数调节**：二值化阈值、最小面积、形态学核大小等
- **颗粒交互**：点击颗粒高亮 + 显示该颗粒的详细参数
- **导出**：标注图 (PNG) + 统计报告 (CSV/PDF)

## 依赖

- Python 3.13
- opencv-python >= 4.8
- ultralytics >= 8.0（YOLOv8）
- PyQt6 >= 6.6
- matplotlib >= 3.8
- numpy >= 1.26
- scipy >= 1.12（统计拟合）

## 导出格式

- **标注图**：PNG，轮廓叠加在原图上
- **统计报告**：CSV（每颗粒一行，各参数为列）+ PDF（含图表摘要）
